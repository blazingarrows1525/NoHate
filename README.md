# 🛡️ NoHate

AI-powered Hate Speech Detection System using NLP

## 🔍 Overview
NoHate is a web-based application that detects hate speech, offensive language, and neutral content using Natural Language Processing techniques.

## 🚀 Features
- Real-time text classification
- Ensemble ML model (SVM + Logistic Regression + SGD)
- NLP Pipeline Visualization
- Batch Prediction Support
- Word Sense Disambiguation (WSD)

## 🧠 Tech Stack
- Python
- Streamlit
- Scikit-learn
- NLTK
- TensorFlow (optional)

## 📊 Classes
- Hate Speech
- Offensive Language
- Neither
